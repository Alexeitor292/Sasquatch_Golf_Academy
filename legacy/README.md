# Legacy Prototype

This folder keeps the original static homepage prototype and its media assets together in one place so the root of the repository stays focused on the Next.js platform app.

Current prototype files live in:

- `legacy/site-prototype/index.html`
- `legacy/site-prototype/styles.css`
- `legacy/site-prototype/script.js`

The prototype is preserved as design/reference material during the platform migration.
